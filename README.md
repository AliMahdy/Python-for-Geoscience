## 📌 About This Repository

This repository accompanies the **"Python for Geoscience"** YouTube series by Dr. Ali Mahdy. It contains practical, reproducible code examples that demonstrate how to apply Python to real-world geoscience challenges — from well-log analysis and seismic processing to rock physics modeling and CO₂ storage assessment.

Whether you're a student learning to code, a researcher automating workflows, or an industry professional exploring data-driven subsurface interpretation, you'll find ready-to-use scripts and clear documentation to accelerate your work.

---

## 🎯 What You'll Find Here

### 🔹 Core Libraries & Tools
- `pandas`, `numpy`: Data manipulation and numerical analysis
- `matplotlib`, `seaborn`, `plotly`: Publication-quality visualization
- `scipy`: Scientific computing and signal processing
- `lasio`, `welly`: Well log data I/O and analysis
- `obspy`: Seismology and time-series processing
- `rasterio`, `geopandas`, `cartopy`: GIS and spatial data handling
- `scikit-learn`, `tensorflow/pytorch`: Machine learning for subsurface modeling

### 🔹 Applied Geoscience Modules
| Module | Description |
|--------|-------------|
| `well_log_analysis/` | LAS file reading, curve conditioning, porosity/permeability modeling |
| `seismic_processing/` | Basic seismic trace handling, filtering, attribute extraction |
| `rock_physics/` | Gassmann fluid substitution, Vp/Vs modeling, elastic moduli calculations |
| `gis_remote_sensing/` | Raster processing, coordinate transformations, map creation |
| `ccus_modeling/` | CO₂ plume visualization, storage capacity estimation workflows |
| `machine_learning/` | Physics-informed ML examples for lithology prediction & property estimation |

### 🔹 Educational Resources
- Jupyter Notebooks with step-by-step explanations
- Sample datasets (synthetic & anonymized field data)
- Setup guides for Anaconda, virtual environments, and package management
- Best practices for reproducible research and version control
